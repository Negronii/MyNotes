# Quantitative Trading 学习笔记

## 量化交易的核心：寻找异象与因子

在量化交易中，一个重要目标是寻找市场中的“异象”（anomaly）：也就是一些在历史数据中反复出现、且不能被传统基本面解释或现有风险模型完全解释的收益模式。

这些异象本身通常不能直接交易，需要被转化成可以量化、可以排序、可以回测的指标，也就是“因子”（factor）。因子可以理解为描述资产某种特征的变量，并且这个变量和未来收益之间可能存在统计关系。

例子：

- 动量异象：过去一段时间涨幅较大的股票，未来一段时间继续表现较好的概率可能更高。它对应的常见因子是动量因子（momentum factor），相关策略通常被称为 momentum strategy。
- 小市值异象：市值较小的股票在历史上可能具有更高的平均收益，这对应市值因子或规模因子（size factor）。
- 价值异象：估值更便宜的股票，例如低市盈率、低市净率或高账面市值比的股票，未来可能有更高的平均收益，这对应价值因子（value factor）。

因此，量化交易的一条核心路径是：

1. 从市场中发现稳定、可解释或可重复验证的异象。
2. 把异象转化为可以量化的因子。
3. 用历史数据检验因子的有效性、稳定性、风险暴露和交易成本。
4. 在控制风险和成本之后，尝试从因子中获得超额收益。

## 因子收益率与因子暴露

对于一个投资组合的收益，可以从因子模型的角度进行拆解：

```text
组合收益 ≈ 因子暴露 × 因子收益率 + 残差收益
```

其中：

- 因子暴露（factor exposure）表示投资组合对某个因子的敏感程度。例如，一个组合是否更偏向小市值股票、价值股、动量股等。
- 因子收益率（factor return）表示单位因子暴露在某段时间内带来的收益。
- 残差收益则是不能被这些因子解释的部分。

在实际研究中，如果因子暴露和因子收益率都未知，确实会出现估计上的相互依赖问题：想估计因子收益率，需要知道资产对因子的暴露；而想准确评估暴露，又可能需要借助收益数据和模型假设。

一种常见做法是先用可观察的特征来定义因子暴露，例如过去收益率、市值、估值指标等，然后按照因子值对股票进行排序。接着，可以做多因子排名靠前、预期收益更高的一组股票，做空因子排名靠后、预期收益更低的一组股票，构造多空组合。

这种做法的目的不是消除所有风险，而是尽量抵消与目标因子无关的市场风险或其他共同风险，让组合收益更集中地反映目标因子的收益来源。

## 多重排序法与双重排序

排序法是因子研究中很常见的一类方法。它的核心做法是先选择一个“排序变量”（sorting variable），用这个变量把股票从高到低或从低到高分组，然后观察不同组的未来收益是否存在系统性差异。

排序变量通常是一个可以在事前观察到的资产特征，例如：

- 市值：可以用来研究小市值股票和大市值股票的收益差异。
- 账面市值比、估值倍数：可以用来研究价值股和成长股的收益差异。
- 过去一段时间的收益率：可以用来研究动量效应。
- 新构造出来的信号：可以用来检验一个新因子是否真的有解释力。

双重排序（double sort）就是同时使用两个排序变量来构造投资组合。例如先按市值分组，再在每一个市值组内部按动量分组；或者分别按市值和动量独立分组，再取两个分组结果的交集。这样可以得到类似 5 x 5 或 2 x 3 的组合矩阵，每一个格子都是一个模拟投资组合。

双重排序有两种常见方式：

- 独立排序：两个变量分别独立排序，然后取交集。例如先把股票分成小市值和大市值，再独立分成低动量、中动量、高动量，最后形成 2 x 3 个组合。
- 条件排序：先按第一个变量分组，再在每一个分组内部按第二个变量排序。例如先分市值组，再分别在小市值组和大市值组内部按动量排序。

双重排序的一个重要用途是控制已有因子的影响。假设研究者发现了一个新因子，但这个新因子可能只是已有因子的一种变形，例如它和市值、价值或动量高度相关。此时可以把新因子和已有因子一起做双重排序：在每一个已有因子的分组内，再观察新因子从低到高的组合收益差异。

如果新因子在控制已有因子之后仍然能带来稳定的多空收益，说明它可能提供了额外信息；如果收益差异明显消失，说明它可能只是重复捕捉了已有因子的影响。

因此，双重排序不是保证“消除暴露”的方法，而是一种控制混杂因素、检查因子独立性、构造因子模拟组合的方法。它可以帮助减少对某些已知因子的非目标暴露，但最终还需要用回归或风险模型进一步检验组合实际暴露。

多重排序则是双重排序的扩展，也就是同时考虑三个或更多排序变量。它能控制更多维度，但会带来组合数量快速增加、每个组合内股票数量变少、估计更不稳定的问题。

## 多因子模型与回归检验

在多因子模型中，一个资产或投资组合的超额收益可以表示为多个因子的线性组合：

```text
R_i - R_f = alpha_i + beta_i1 * F_1 + beta_i2 * F_2 + ... + beta_ik * F_k + epsilon_i
```

其中：

- `R_i - R_f` 是资产或组合的超额收益。
- `F_1 ... F_k` 是多个因子的收益率，例如市场因子、规模因子、价值因子、动量因子等。
- `beta` 是资产对每一个因子的暴露。
- `alpha_i` 是定价误差，也可以理解为模型无法解释的平均超额收益。
- `epsilon_i` 是残差项。

如果一个多因子模型能够很好地解释资产收益，那么模型留下的定价误差 `alpha` 应该足够接近 0。更重要的不是单独看某一个资产的 alpha 是否为 0，而是联合检验所有测试资产的 alpha 是否在统计意义上同时接近 0。

一个简单的回归检验流程可以理解为：

1. 选定测试资产或测试组合，以及要检验的因子收益率。
2. 对每一个测试资产分别做时序回归，估计它对各个因子的暴露 `beta`，以及模型留下的定价误差 `alpha`。
3. 对所有测试资产的 `alpha` 做联合检验，判断这些定价误差是否整体上可以被认为接近 0。

时序回归是在时间维度上进行的：对同一个资产或组合，观察它在很多期里的收益如何随因子收益变化。它常用于估计资产的因子暴露和 alpha。

截面回归是在资产维度上进行的：在同一个时间点或一段时间内，观察不同资产的平均收益是否可以被它们的因子暴露解释。Fama-MacBeth 回归就是经典的截面资产定价检验方法之一。

因此，时序回归和截面回归都可以用于股票和投资组合研究。它们的区别不是“股票用时序、非股票数据用截面”，而是研究问题不同：时序回归更关注单个资产随时间变化的因子暴露和 alpha，截面回归更关注不同资产之间的收益差异是否能被风险暴露或特征解释。

## 当前理解

量化交易不是简单地寻找“会涨的股票”，而是系统性地寻找市场中可以被数据验证的收益模式。异象提供研究方向，因子提供量化表达，排序、多空组合、回归检验和风险控制则是把因子转化为策略的重要步骤。
